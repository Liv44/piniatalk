## Launch App

1. Installer les dépendances 🔄

`npm install` ou `yarn install`

2. Lancer le projet ▶️

`npm run dev` ou `yarn dev`

3. Aller sur l'URL http://localhost:5173

4. Se connecter avec son nom d'utilisateur et son mot de passe 🔒

5. Enjoy ! 🎉
